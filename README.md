# Online High School Exam Preparation Website

## 1. Project Overview
This project is an online quiz-based learning platform designed to support Vietnamese high school students in preparing for national examinations based on the **new high school curriculum**. The system provides practice tests for all major subjects and integrates an AI assistant to help students answer questions and clarify doubts during their learning process.
